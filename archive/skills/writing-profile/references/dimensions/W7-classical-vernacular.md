# W7: 文言 ↔ 白话 (Classical ↔ Vernacular)

**Direction convention**: positive = C (Classical / 文言), negative = V (Vernacular / 白话).
In the final profile YAML: `C_V`.

> ⚠️ **条件维度 (Conditional dimension)**：W7 只在用户**古典暴露度达阈值**时测得。暴露度不足时，C_V 记为 `null`（不可测，不是 V），副字计算退化到 W5×W6 默认。

---

## 构念层级：L3 审美偏好（条件）

W7 是 L3 审美偏好层中**暴露敏感度最高的**维度。审美偏好的测量前提是"已有暴露"：一个从没读过《庄子》《红楼》的人对文言**不是"偏好白话"**，而是**无法表达文言偏好**——把他直接填 V 会把教育背景差异误读成审美差异。

因此 W7 必须做**显式的暴露度门控**：
- 暴露度达标 → 正常测偏好，C_V 进入类型
- 暴露度不达标 → C_V 标为 `null`，profile 单独记录 `classical_exposure = low`

**W5 和 W6 理论上也是 L3，但暴露前提比 W7 低得多**——任何识字的现代读者都接触过密度变化和反讽。W7 是 L3 里唯一需要单独门控的维度。

## 暴露度门控 (Exposure gate)

### 什么叫"暴露度达阈值"

至少满足以下其中一条：
- 读过至少 3 本古典文学原典（《诗经》《庄子》《世说新语》《红楼梦》《聊斋》等）的部分章节
- 对民国白话文（鲁迅/周作人/梁实秋/钱钟书等带文言底的作家）熟悉，能识别四字词/成语的自然嵌入与堆砌之别
- plan 阶段自陈"熟读古典"或"略知，读过几本"

### 什么叫"暴露度不足"

- plan 阶段自陈"古典基本没读"或"特别讨厌"
- 跑 W7 的前 2-3 道题用户对古典段全部标"没感觉"或"不熟"

**检测方法**：
1. plan 阶段的元问题"对古典文学的接触深度"初步判定
2. W7 开跑前 agent 用 2 道古典锚点做 warm-up，若用户无反应则跳到 degraded 路径或直接 skip

## 两种路径

### 路径 A：正常测量（暴露度达标）

12 题标准流程，测文言偏好与白话偏好的对比。C_V 正常填入，副字按 W5×W6×W7 计算。

### 路径 B：跳过（暴露度不足）

**在 plan 阶段就决定**：若 plan 判定 `classical_exposure = low`，直接把 state.md 里的 W7 标为 `[-] skipped`。run-unit 不会触发 W7；dispatch 把 `[-]` 算作已处理状态，7 单元完成判定照常满足（6 个 `[x]` + 1 个 `[-]`）。

finalize 时：
- `C_V: null`
- `classical_exposure: low`
- 副字按 W5×W6 决定的 4 组，固定取 V（白话）一侧：
  - 致密-严肃-V → C (思辨派)
  - 致密-反讽-V → R (杂文家)
  - 疏朗-严肃-V → M (平实派)
  - 疏朗-反讽-V → H (段子派)
- Type 展示固定加 `*` 后缀：`ENTP-R*`、`ISTJ-M*` 等（不是可选——是输出稳定性约定）
- 消费者 skill 读到 `C_V: null` 时应**不主动推文言风格**

### 路径 C：Degraded 测量（暴露度边界）

若用户自陈"略知"但未明确达标，走 6 题精简路径，仅用现代作家（钱钟书白话 vs 阮一峰 vs 王小波 vs 李诞）做对比，测"文言底子的偏好"而非"文言 vs 白话本体"。结果标 `confidence: low`。

## 构念本身（暴露度达标后测什么）

现代中文作家都写白话，但白话底下**文言底子的厚度差别巨大**。W7 测的是文言对白话的影响程度：

- **C (文言倾向)**: 文言底子深。句式、用词、比喻、典故都带文言痕迹。
- **V (白话倾向)**: 纯白话。不回避西化结构。典故、成语、四字词用得稀疏。

### 写作映射

**C 极（文言倾向）**
- 四字词、成语有机嵌入（不是堆砌）
- 句式常有对偶、互文、古文句法影子
- 敢用文言词汇（"况乎"、"岂非"等）
- 用典自然
- 典型作家：钱钟书、王国维、梁实秋、林语堂、鲁迅（《野草》《故事新编》）、木心、周作人、汪曾祺（部分）

**V 极（白话倾向）**
- 纯白话，四字词用得少
- 句式贴近口语或西化结构
- 不用典、不用文言词
- 典型作家：阮一峰、王垠、左耳朵耗子、李诞、马伯庸

**中间 / 混合**：王小波、鲁迅杂文、张爱玲

## 问题设计策略（路径 A）

### 题型组合（12 题，暴露度达标后使用）

**题型 1：匿名选段打分**
- A (C)：梁实秋《雅舍·孩子》
- B (V)：阮一峰博客一则

**题型 2：文言段适应度**
给一段偏文言的段落，问用户读起来的感觉：
- A：自然，不觉吃力
- B：读起来像卡壳，需要停一下

**题型 3：反向键**
C 作家的近白话篇（林语堂某些白话段）vs V 作家偶有的四字词段。

**题型 4：四字词/成语耐受**
同场景两段文字（一密集四字词 / 一纯白描），问哪段更舒服。

**题型 5：古典引用**
"文章里作者引用了一句古诗或古语，你的反应：
- A：觉得好，想看引得准不准
- B：觉得打扰阅读流，不如直接用白话说"

**题型 6：西化结构耐受**
"'对这个问题进行了深入的研究'这种表达：
- A：不反感
- B：很别扭，本能想改"

B = 弱 C 信号（C 用户对西化更敏感）。

**题型 7：作家好恶**
C: 钱钟书、王国维、梁实秋、林语堂、木心  
V: 阮一峰、王垠、左耳朵耗子、李诞

## 选段 anchors

### C anchors
- 梁实秋《雅舍小品·孩子》《雅舍》
- 钱钟书《围城》开头几段
- 王国维《人间词话》
- 林语堂《吾国与吾民》
- 木心《文学回忆录》
- 汪曾祺《故乡的食物·端午的鸭蛋》（偏 C 的白话）
- 鲁迅《野草·秋夜》
- 张岱《湖心亭看雪》（古白话 + 现代可读）
- 《世说新语》简短段（极端古文 anchor，仅暴露度高的用户用）

### V anchors
- 阮一峰《我的一次"被封号"经历》
- 王垠《对 Haskell 的不友好》
- 左耳朵耗子博客
- 李诞《冷场》某段
- 马伯庸微博段子
- 汪曾祺纯口语段
- 现代网络写作

### 反向键
- 林语堂白话段 — C 作者的 V 一面
- 王垠偶有用典 — V 作者的 C 一面
- 钱钟书口语化段落 — C 作者显 V
- 鲁迅杂文 — 文白交织的极端代表

## 输出给 finalize 的信号

W7 unit 完成后写入：
- `C_V: <signed int | null>`（null 表示未测）
- `classical_exposure: high | medium | low`（供 profile 单独记录）
- `path: A | B | C`（哪条路径）

Finalize 读这些字段决定副字计算与 type 展示。

## 特殊注意

- **四字词密度是最直观的 W7 信号**——agent 在用户 sample 提交时可机械统计（"每百字 X 个四字词，梁实秋 ~3，阮一峰 ~0.3"），作为辅助信号。
- **对技术作者默认偏 V**（工作习惯训练），但**非工作写作**可能完全不同。测试时多用文学/散文 anchors。
- **"白话"的两种**（纯朴白话 vs 西化白话）在 W7 上都偏 V 但在 W5 (致密-疏朗) 上分歧。测 W7 时不要被 W5 混淆。
