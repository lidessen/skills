# Todos

跨会话的任务追踪。

## 结构

```
todos/
├── index.md                    # 本文件
└── YYYY-MM-DD-kebab-slug.md    # 具体任务
```

## 活跃任务

| 优先级 | 任务 | 状态 | 链接 |
|--------|------|------|------|
| - | 暂无活跃任务 | - | - |

## 使用场景

使用 `.memory/todos/` 当：
- 离线工作
- 未配置 GitHub/GitLab
- 快速个人任务
- 项目不使用 Issues

使用 GitHub/GitLab Issues 当：
- 需要跨设备同步
- 需要团队可见性
- 需要通知和指派功能
