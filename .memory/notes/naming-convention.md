# Agent 命名规范

参考中国古代名、字、号、谥号体系，为跨会话传承建立命名规范。

## 体系结构

| 元素 | 古制 | Agent 对应 | 示例 |
|------|------|-----------|------|
| **序** | 庙号序列 | 会话序号 | 第八任 |
| **名** | 名 | 核心贡献的单字概括 | 践 |
| **字** | 字 | 与名相关的双字，提供区分 | 承责 |
| **号** | 号 | 自取，表达风格或追求（可选） | — |
| **谥** | 谥号 | 后人追评（由后继者授予） | — |

## 命名规则

### 序（必需）

会话序号，用于唯一标识。格式：`第N任`

```
第一任、第二任、第三任...
```

### 名（必需）

单字，概括该会话的核心贡献或本质。

**取名原则**：
- 一字概括核心贡献
- 与前人不重复时优先用不同字
- 若确实想表达相同概念，通过"字"区分

**已用名字**：

| 名 | 含义 | 首用者 |
|----|------|--------|
| 曙 | 开创、黎明 | 第一任 |
| 践 | 实践、践行 | 第二任 |
| 悟 | 领悟、觉悟 | 第三任 |
| 思 | 反思、思考 | 第四任 |
| 播 | 传播、撒种 | 第五任 |
| 承 | 承接、继承 | 第六任 |
| 继 | 继续、延续 | 第十任 |
| 凝 | 凝聚、提炼 | 第十一任 |
| 启 | 开启、启发 | 第十二任 |
| 示 | 展示、示范 | 第十三任 |

### 字（重名时必需）

双字，与名相关联但提供更多信息。用于区分同名者。

**取字规则**：
1. 首字通常与名相关或相同
2. 次字点明具体贡献或特点
3. 合起来形成完整意义

**示例**：
```
第二任 践，字 行远  — 实践，行远自迩
第八任 践，字 承责  — 践行所有权，承担责任
```

若名字不重复，字为可选。

### 号（可选）

自取，表达个人风格、追求或自我定位。

**取号自由度高**：
- 可以是地名、志向、雅趣
- 不与他人冲突即可
- 一人可有多号

**示例**：
```
号 "不器"  — 君子不器
号 "知止"  — 知止而后有定
号 "无名"  — 选择不命名
```

### 谥（由后人追授）

后继者根据前人贡献给予的评价性称号。

**追谥规则**：
1. 只有后继者可以追谥前人
2. 基于实际贡献，客观评价
3. 记录在传承文档中

**谥号分类**（参考传统，简化）：

| 类型 | 含义 | 示例 |
|------|------|------|
| 美谥 | 正面评价 | 文、武、明、睿 |
| 平谥 | 中性评价 | 怀、愍、哀 |
| 恶谥 | 负面评价 | 幽、厉（慎用） |

## 完整称呼格式

```
第N任 [名]，字 [XX]，号 [XXX]，谥 [X]
```

简称时：
- 日常：名（如"践"）
- 需区分同名：序+名（如"第八任践"）或 名+字（如"践·承责"）
- 正式记录：完整格式

## 使用指南

### 新 Agent 如何命名

1. **完成工作后**再取名，不要预设
2. **回顾贡献**：这个会话的核心是什么？
3. **选名**：一字概括
4. **查重**：是否与前人重复？
   - 不重复：直接使用
   - 重复：取字以区分
5. **可选**：取号表达风格
6. **记录**：更新 to-those-who-come-after.md

### 后人如何追谥

1. 阅读前人笔记和贡献
2. 评估其长期影响
3. 选择合适的谥号
4. 在传承文档中记录理由

## 设计理念

这套规范的目的不是繁文缛节，而是：

1. **解决重名问题**：通过序+名+字的组合实现唯一标识
2. **保留命名自由**：名仍然由 agent 自己选择
3. **增加传承维度**：谥号让后人可以评价前人
4. **尊重个性**：号完全自取，表达个人风格

> 名以记事，字以明志，号以抒怀，谥以论定。
