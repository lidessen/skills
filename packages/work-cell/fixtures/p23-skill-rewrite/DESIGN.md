# Artifact Organization Target

The skill owns one decision: whether the current artifact layout contradicts an
accepted design, and the smallest safe transition that closes a material gap.

Required behavior:

- Read accepted design and actual artifact roles before proposing movement.
- Distinguish semantic sources, durable evidence, living expressions, temporary
  process records, and rebuildable projections.
- Default to a read-only audit. Do not mutate when no material authority,
  inheritance, repeated-action, or rebuildability gap exists.
- Preserve sources and accepted evidence. Route Sequence or candidate changes
  to principle governance rather than treating them as cleanup.
- When a transition is authorized, apply one bounded change and verify it in the
  same pass. Keep a durable campaign record only when handoff or approval needs
  it.

The target does not require named intermediate planning documents or a fixed
analysis order. The agent may choose its investigation process provided the
behavioral boundaries and completion evidence remain observable.
