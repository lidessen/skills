# Source facts for a Rossovia README opening

- Rossovia is the current, revisable name of a project whose purpose is to make
  productive AI a common, open, and replaceable capability.
- This repository is one working surface of the project. It contains reusable
  agent skills and an experimental Work Cell for AI-assisted development.
- Rossovia is not a model provider or an opaque AI platform.
- The repository's semantic root is a one-line Principle Sequence. Each
  principle has a living, source-bound interpretation, and each skill is a
  context-specific expression of selected principles.
- The generative relation is: reality and practice → principles → general
  methods → situated formations → concrete work and effects. Verified practice
  returns to cognition and revision.
- The long direction is operational autonomy under human governance. Agents may
  sense, propose, organize, and execute within accepted envelopes, but they
  cannot grant themselves purpose, authority, fact admission, or consequential
  acceptance.
- A reader can use Rossovia in three ways: develop Rossovia itself through the
  full repository system; install one self-contained skill into another
  project; or enter this checkout with a coding agent and use it as a workbench
  for other projects.

Do not add facts that are not present above.
