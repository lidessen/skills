# Aesthetic Practice

**Status:** pilot
**Authority:** [decision 017](../decisions/017-aesthetic-practice-pilot.md)

This is a small practice area for making the project's productive, open, and
non-exploitative purpose perceptible in documents, interfaces, illustration,
and other public expressions. It is not a brand book, asset warehouse, visual
canon, or substitute for human judgment.

## Inheritance ladder

| Artifact | What it retains | Authority boundary |
|---|---|---|
| `seed/` | explicit Principal preferences and disconfirming contrasts | preference signal, not a shared direction |
| `cases/` | a task's audience, relation, sources, brief, and acceptance | proposed direction for one task |
| `studies/` | observed artifact-in-use, review, failure, and revision | evidence, not a reusable rule |
| `directions/` | a repeated, reviewed relation or pattern | accepted project guidance; not a Sequence source |
| `references/` | attributed external/source material and license/use limits | evidence for a case, never style authority |

Create a subdirectory only when its first real artifact exists. Do not stockpile
images or copied styles. Every retained reference must name what decision it
changed, its source/license, and what must not be copied.

## Practice loop

1. Start from a real audience, production object, and intended action.
2. Prepare an Aesthetic Case: relation, contrast, few references, material and
   accessibility constraints, negative boundary, and review question.
3. Lower it to a Work Cell Expression Brief only with actual tools/capabilities.
4. Evaluate the expression in use, retain a study, and route failure to the
   next smallest revision.
5. Promote only repeated, human-reviewed lessons to `directions/`.

Project-specific direction comes first. Use mature external design systems and
editorial guides as attributed fallback references, adapting their structures to
this project's users and mandate rather than copying their aesthetic surface.

The first retained sources are the [initial triptych](references/2026-07-10-initial-triptych.md)
and [Common Workshop masthead](references/2026-07-10-common-workshop-masthead.md).
They record Principal preference and accepted seed material, not a public visual
direction.
