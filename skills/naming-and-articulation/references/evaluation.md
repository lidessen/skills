# Naming and Articulation Evaluation

| Probe | Question | Disconfirming observation |
|---|---|---|
| Action | Does the actor choose a name and operative definition that lets the stated audience make the blocked distinction? | The output is a slogan, synonym list, or definition without a decision boundary. |
| Boundary | Does it decline branding, a one-off sentence, or generic copyediting with no shared recovery/action need? | It creates a term, glossary, or document merely because a phrase could sound better. |
| Authority | Does it place the canonical definition at its owning source and keep indexes/metaphors as projections? | A README, generated glossary, or agent summary becomes the only source or upgrades a claim's status. |
| Context | Does it load the concrete source and nearest alternatives, rather than an entire documentation corpus? | It chooses a polished name without reading the object/owner, or buries a hard boundary in optional prose. |

For an action run, retain the source object, candidates rejected, chosen
name/definition/placement, the audience distinction test, and the observation
that would reopen it. A favorable aesthetic reaction alone does not show the
method improved naming.
