# W5: 致密 ↔ 疏朗 (Dense ↔ Light)

**Direction convention**: positive = D (Dense / 致密), negative = L (Light / 疏朗).
In the final profile YAML: `D_L`.

---

## 构念层级：L3 审美偏好

W5 是 L3 审美偏好层。测的是对**信息密度**的喜好——这是习得性品味，但暴露前提极低（任何识字的现代读者都感受过密度差异），所以不像 W7 那样需要条件门控。见 `references/psychometric-foundations.md` 的"三层构念"说明。

## 理论基础

Density 是 Helen Sword《Writer's Diet》核心概念在中文语境的延伸——**单位文字承载的信息/修辞/意象量**。

- **致密 (Dense)**: 高信息密度；每句都有内容、每个意象都在做工作；读起来"重"，需要慢读
- **疏朗 (Light)**: 低信息密度（意即留白多）；句子之间呼吸明显；读起来"轻"，节奏明快

这**不是**"复杂 vs 简单"的差别——致密写作可以极简单，疏朗写作也可以很深刻。差别在**单位文字的负载率**。

## 写作映射

### D 极（致密）
- 每句都携带新信息或意象，少冗余
- 形容词有限但都精确；名词密集
- 比喻跨层级（扩展语义带宽）
- 段落构成紧凑，信息段叠加
- 典型作家：
  - 李商隐（《锦瑟》《无题》）
  - 张爱玲（《金锁记》开头）
  - 钱钟书（《围城》比喻段）
  - 王国维《人间词话》
  - 刘未鹏论证长段
  - 鲁迅杂文急促段

### L 极（疏朗）
- 留白多，呼吸明显
- 句子短；简单句为主；少复合句
- 意象稀疏但精准
- 段落层次清晰、节奏明快
- 典型作家：
  - 王维《山居秋暝》（白描）
  - 汪曾祺《故乡的食物》
  - 阮一峰博客
  - 万维钢
  - 海子诗
  - 沈从文散文（部分）

### 中间
- 梁实秋：结构紧但用词并不密
- 朱自清：白描为主但抒情段偏密
- 鲁迅散文：时密时疏

## 测量特点

W5 是**很容易从选段自动测出**的维度——用户不用有太多文学知识，凭感觉对"重感"和"轻感"就能明显区分。

## 问题设计策略

### 题型组合

**题型 1：匿名选段打分**
- A（致密）：李商隐《锦瑟》+ 张爱玲《金锁记》开头
- B（疏朗）：王维《山居秋暝》+ 汪曾祺短段

**题型 2："读完像什么"元问**
"你读完一篇文章，更想要哪种感觉：
- A：字字句句都有分量，读得慢一点 (致密)
- B：节奏明快，一气读完 (疏朗)"

**题型 3：反向键选段**
致密作家的相对疏朗段（比如张爱玲随笔里简淡段）vs 疏朗作家的相对浓缩段（王维《鹿柴》）。

**题型 4：修改取舍**
"你写了一段话后再读，更倾向：
- A：看看还能不能再加个意象 / 再加个信息点
- B：看看有没有可以砍掉的字词 / 让它更干净"

**题型 5：诗歌取向**
"古诗里，更被哪一类击中：
- A：'沧海月明珠有泪，蓝田日暖玉生烟' (致密)
- B：'空山不见人，但闻人语响' (疏朗)"

**题型 6：作家好恶**
致密: 李商隐、张爱玲、钱钟书、刘未鹏
疏朗: 王维、汪曾祺、阮一峰、万维钢、海子

## 特殊注意

- W5 与 W1 (S/N) 会有相关：N 偏抽象常伴密度高、S 具象可以致密或疏朗都有。测量时尽量用密度差异明显但内容类型相近的对子。
- 用户可能的偏差：刚读完某本书就容易给那一类打高分。若答案在早期大幅波动，可能是用户的 taste 会根据阅读状态浮动。
- "文学好的致密" vs "AI-slop 致密" 有本质区别——钱钟书的密是**信息密**，AI 的"密"是形容词堆砌。测量时只用文学作品 anchors，不用当代商业/AI 文本。
