# W4: Judging ↔ Perceiving (收束 ↔ 发散)

**Direction convention**: positive = J (judging / 收束), negative = P (perceiving / 发散).

> ⚠️ 本维度借用 MBTI J/P 字母但**构念已重定义为写作专属**：测的是**文本的收束度**（封闭/开放），与 MBTI 人格 J/P 测生活方式不可等价。见 `references/psychometric-foundations.md` 的"三层构念"说明。

---

## 构念层级：L2 文本姿态

本维度是文本姿态层（text-stance layer），与 W3 并列。测量对象是**成文的结构收束度**——不是作者起稿流程，也不是作者生活方式。

**严守边界**：
- ❌ 不测"大纲先行 vs 边写边想"（那是起稿流程，属于 L0 工法层；INFP 也可以列大纲、ENTJ 也可以乱写）
- ❌ 不测"你写作时是否有计划"（工法，不测）
- ✅ 只测成文的结构特征：文本是否封闭、结论是否明确、段落之间是否承接严整

## 理论基础

MBTI J/P 原指生活方式（结构化 vs 弹性）。**在写作上**重诠释为**成文的结构处理**：

- **J (收束型)**: 封闭结构。明确论点、清晰脉络、确定结论。读完知道作者想说什么。段落承接严整，不放漫游。
- **P (发散型)**: 开放结构。主题可能有多个线头、允许枝蔓、结论开放或干脆不给。读完留有余韵或悬而未决。段落可以跳脱、回旋。

这**不是**"思路清楚 vs 思路混乱"——P 型作者思路可以极清楚，只是他们选择写的时候保留可能性、不强行封闭。

## 写作映射

### J 极
- 论点：明确提出、全文扣紧
- 段落：承接严整，每段一个命题，段末常有收束
- 结论：显性、可一句话复述
- 典型作家：
  - 胡适
  - 万维钢
  - 吴军
  - 刘未鹏
  - 鲁迅杂文（大部分）
  - 李泽厚学术

### P 极
- 论点：可能没有单一论点，或让论点从素材中慢慢浮现
- 段落：可以跳脱、自由联想、从 A 聊到 B 再回到 A
- 结论：开放、多义、或以反问/抒情收束
- 典型作家：
  - 蒙田（中译）
  - 周作人
  - 汪曾祺
  - 钱钟书《围城》（小说 P，论说里也有 P 特征）
  - 木心《文学回忆录》
  - 王小波（有 P 气息的论说）
  - 李诞短文

### 中间
- 鲁迅：杂文 J，散文诗 P（《野草》）
- 刘瑜：早期 J，后期部分篇 P
- 钱钟书：学术 J、小说 P

## 测量特点与常见陷阱

**防串轴**：
- **和 W5 (密度) 串**：J 常伴密（结构严 + 信息密），但梁实秋是 J+疏（结构封闭但用词闲适）、汪曾祺是 P+疏。要分开测。
- **技术写作基线偏 J**：技术文章天然追求结论清晰。对技术作者测 W4 时，**务必多用散文/文学 anchors**，测他们**非工作写作**时的本心偏好。
- **误解"P = 没逻辑"**：P 文本可以结构严密，只是选择不封闭结论或不给读者一句话概括。

## 问题设计策略

### 题型组合（12 题）

**题型 1：匿名选段打分**
- A (J)：万维钢一则（论点-论证-结论结构）
- B (P)：周作人一则（散散地聊，结尾若无其事）

**题型 2："读完什么感觉"元问**
"你读完一篇好文章，更希望：
- A：合上时能一句话总结它在说什么
- B：合上时脑子里还有余韵、有没说完的东西"

**题型 3：反向键选段**
J 作家的 P 篇（鲁迅《野草》）vs P 作家的 J 篇（钱钟书学术段）。

**题型 4：结尾风格**
给一个假设文章开头，让用户选择倾向哪种结尾：
- A (J): "综上所述，关键在于 X。"式清晰收束
- B (P): 回到开头的意象、或以反问 / 悬念结束

**题型 5：结构选择**
对同一主题（例：写"孤独"），让用户选：
- A：列点分述（孤独的三种类型）
- B：从一个具体片段出发漫谈

**题型 6：作家好恶**
J: 万维钢、刘未鹏、胡适、李泽厚  
P: 周作人、汪曾祺、蒙田、木心、钱钟书《围城》

### 12 题配比（注意：全部围绕"文本形态"，无起稿流程题）

| Slot | 题型 |
|------|------|
| 1-2 | 匿名选段 |
| 3 | 元问 "读完什么感觉" |
| 4 | 反向键选段 |
| 5 | 结尾风格 |
| 6 | 作家好恶 |
| 7 | 匿名选段 |
| 8 | 结尾风格（不同情境） |
| 9 | 反向键 |
| 10 | 结构选择 |
| 11 | 匿名选段 |
| 12 | 终确认 |

## 选段 anchors

### J anchors
- 胡适《文学改良刍议》（八事分述结构）
- 万维钢《万万没想到》一则
- 吴军《数学之美》某章
- 鲁迅《拿来主义》
- 刘瑜《民主是个好东西》
- 朱自清《匆匆》（虽然抒情但结构严）

### P anchors
- 蒙田《随笔集》任一篇（跳脱的典范）
- 周作人《乌篷船》（漫谈式）
- 汪曾祺《花》
- 钱钟书《围城》某段跳脱的叙述
- 木心《文学回忆录》某些片段
- 张岱《陶庵梦忆》各小品
- 李诞短段子

### 反向键 anchors
- 鲁迅《野草》 — P 散文诗 + 背后 J 骨架
- 钱钟书学术段 — 表面漫议但结构紧
- 梁实秋《雅舍小品》— 看似闲适漫谈其实 J 结构

## 与 W5 的边界

W4 和 W5 (致密-疏朗) 最容易混淆——密度和结构是**独立维度**：
- 梁实秋：D 低 J 高（行文闲适但结构封闭）
- 汪曾祺：D 低 P 高（行文闲适且结构发散）
- 刘未鹏：D 高 J 高（密 + 封闭）
- 钱钟书《围城》：D 高 P 倾向（密 + 漫议）

测量时用同密度的选段对比，让收束度成为唯一差异。

## 关于"中间型"

这个维度上"中间型"相对常见——不少作者是根据主题选择结构，本身没有一贯偏好。中间分结果要认真报告为 flexible，而不是误判或强推一侧。
