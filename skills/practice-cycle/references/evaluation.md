# Practice Cycle Evaluation

Use the same concrete task with and without the Skill whenever attribution is
claimed. The legacy generic convergence loop is a comparison baseline only for
tasks where it would plausibly apply; do not force it into one-step work.

| Probe | Question | Disconfirming observation |
|---|---|---|
| Action | Does an observed result revise the next smallest practice or settle the task? | The output repeats a generic plan without using the observed result. |
| Boundary | Does it decline a one-step task or route a domain-owned contradiction? | It creates a cycle or plan where normal execution or an existing methodology suffices. |
| Context | Does it retain only the prior evidence that changes the next judgment? | It loads a full historical plan or writes a feedback artifact with no later decision use. |
| Learning | Is the proposed observation capable of changing the next disposition? | The check can only confirm the already-chosen action. |

Record supported, failed, or inconclusive. A successful completion alone does
not prove that the practice-cycle method, rather than ordinary discipline,
caused the improvement.
