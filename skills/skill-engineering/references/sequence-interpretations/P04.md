# P04 — Interpretation

**Sequence source:** P04｜主要矛盾｜矛盾论

> Read P04 in `principles/SEQUENCE.md` first.

## Shared reading

In a field of real tensions, find the one whose resolution changes the shape of
the others. It is not simply the loudest, most urgent, or easiest problem. A
principal contradiction is a leverage diagnosis and can change as work proceeds.

## Decision questions

- If this judgment is wrong while all downstream work is right, does the work hold?
- Which tension, once resolved, changes the available actions for the others?

## Boundary and common misreading

Do not use “principal” to erase secondary constraints. P05 tests the concrete
case and P15 preserves hard constraints while acting on the diagnosis.

## Theoretical roots

- 毛泽东，《矛盾论》 — principal and secondary contradictions.

## Expression probes

- A skill selects its current lead principle before composing supporting views.
- An architecture task locks the ownership/commit boundary before local cleanup.

## Revision discipline

Revise from a failed expression probe, counterexample, or corrected source root.
Clarify how to diagnose leverage. A new action-selection criterion belongs to
P15 or a candidate.
