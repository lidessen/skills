# Principle Sequence

P01｜原则高于规则｜一般化判断
P02｜实事求是｜唯物主义认识论
P03｜实践—认识—再实践｜实践论
P04｜主要矛盾｜矛盾论
P05｜具体问题具体分析｜矛盾特殊性
P06｜本质与现象｜辩证法
P07｜由具体经抽象再上升为具体｜马克思主义哲学
P08｜可证伪性｜科学哲学
P09｜注意力分层｜认知与信息架构
P10｜有限人类带宽｜人机协作
P11｜决策权与执行权分离｜组织设计
P12｜外化记忆与代际继承｜分布式认知
P13｜主张不等于事实；事实须经可追溯的验证提交｜实践论 / 控制论
P14｜可由源重建者只是投影，不得拥有事实权｜系统论 / 事件溯源
P15｜只选择化解当前矛盾且保留硬约束的最小有效跃迁｜控制论 / 动态规划
P16｜表达形式须使实践主体能够行动；形式错位，正确内容亦会失效｜辩证法 / 实践论
