# P09 — Interpretation

**Sequence source:** P09｜注意力分层｜认知与信息架构

> Read P09 in `principles/SEQUENCE.md` first.

## Shared reading

Attention is not flat. Stable, high-leverage orientation belongs in a small
always-present layer; task-specific method belongs on activation; volatile detail
belongs on demand. The point is not hiding information but presenting it at the
layer where it can affect the current judgment.

## Decision questions

- What must be known before any action, and what is only needed for this case?
- Does this detail change the current decision enough to occupy active context?

## Boundary and common misreading

Layering is not an excuse to bury a hard constraint. A safety or authority
condition remains discoverable and available when it governs the task.

## Theoretical roots

- Cognitive and information architecture; progressive disclosure in the Agent
  Skills Specification.
- Stewart Brand's pace-layer framework is a supporting analogy for artifacts
  that operate at different speeds: fast layers supply learning and variation,
  while slow layers preserve continuity and constraints. It does not make P09 a
  general rule for module dependency direction.
- Project evidence: L1/L2/L3 harness architecture.

## Expression probes

- `SKILL.md` carries the core method while detailed protocols live in references.
- A review team reads only interpretations for selected P-IDs.
- Fast-loop tests and traces may challenge a slow-loop design artifact, but one
  observation does not silently rewrite the durable layer.

## Revision discipline

Revise from a failed expression probe, counterexample, or corrected source root.
Clarify which layer owns a kind of information. A new human-capacity constraint
belongs to P10.
